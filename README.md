# MM_2do_InGrid
Identification of mortality risk subgroups of BRAF V600E mutated patients

This code was created to identify subgroups of patients with different mortality risk rates within a cohort of patients with BRAF mutation. Details about the INGRID R package utilized for the analysis can be found on:
https://doi.org/10.1177/0962280217752980
https://dongjunchung.github.io/INGRID/
